# Design docs

Internal design documentation for panopticon. User-facing docs live one level up in
[`../`](..) (e.g. `auth.md`, `macos-setup.md`).

- [`GOALS.md`](GOALS.md) — motivation, non-goals, and the milestones.
- [`PARITY.md`](PARITY.md) — feature inventory mined from the cloude-cade prototype.
- [`ARCHITECTURE.md`](ARCHITECTURE.md) — target design: topology, the determinism invariant,
  interfaces, and the milestone map.
- [`ROADMAP.md`](ROADMAP.md) — the vertical slices toward the milestones.
- [`BACKLOG.md`](BACKLOG.md) — incidental findings logged out of slices, rather than expanding
  their scope.
- [`decisions/`](decisions/) — architecture decision records (ADRs).
